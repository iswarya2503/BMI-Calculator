# BMI Calculator

## Description
A Python Tkinter application that calculates Body Mass Index (BMI).

## Features
- Calculate BMI
- Displays BMI category
- Simple graphical user interface

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install the required package:
   ```
   pip install -r requirements.txt
   ```
2. Run:
   ```
   python bmi_gui.py
   ```
