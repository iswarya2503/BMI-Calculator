# BMI Calculator

## Description
A desktop application developed in Python using Tkinter that calculates Body Mass Index (BMI) and stores user records.

## Features
- Calculate BMI
- Display BMI Category
- Color-coded BMI Result
- Save Records to CSV
- View Saved Records
- Show BMI Graph using Matplotlib
- Error Handling

## Technologies Used
- Python
- Tkinter
- CSV
- Matplotlib

## Project Structure

BMI_Calculator/
│── bmi_gui.py
│── README.md
│── requirements.txt
│── bmi_records.csv

## How to Run

1. Install Python
2. Install required library

pip install matplotlib

3. Run

python bmi_gui.py
